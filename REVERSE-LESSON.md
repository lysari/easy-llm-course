# Reverse Lessons — From Meaning to Zero

> Normal lessons go **bottom-up**: from numbers → neural networks → GPT-4.
>
> Reverse lessons go **top-down**: from what everyone SEES → to what's actually happening → down to zero.

---

## The question these lessons answer

Everyone knows AI generates meaningful text.

Almost nobody asks: **does "generating meaningful text" require understanding?**

These lessons answer that question by peeling back the layers one by one — starting from the surface (human-readable output) and going down until there is nothing left but floating-point numbers multiplied together.

By the end, you will see clearly: **the meaning was never in the model. You brought it.**

---

## The direction

```
RL-00  The Illusion            ← start here: what you SEE
RL-01  Text Is Numbers
RL-02  Meaning Is Position
RL-03  Attention Is Arithmetic
RL-04  Prediction, Not Knowing
RL-05  Weights, Not Wisdom
RL-06  Ground Zero             ← end here: what's ACTUALLY THERE
```

Each lesson:
1. **Shows** something that looks like understanding
2. **Questions** whether it is understanding
3. **Reveals** what's actually happening one layer down
4. **Demonstrates** it with runnable TypeScript code
